# Rental SEO Boost

CLI tool to analyze and boost SEO for vacation rental websites. Helps vacation rental owners reduce dependency on Airbnb by improving their direct booking website's visibility.

## Problem It Solves

Vacation rental owners spend money building their own booking websites to avoid Airbnb's 15-20% commission fees, but their sites get zero traffic because they don't understand SEO, advertising, or how to get found on Google.

## Features

- **SEO Health Check**: Analyze meta tags, titles, keyword density, and mobile optimization
- **Local SEO Optimization**: Get suggestions for Google Business Profile, local keywords, and map integration
- **Content Marketing Plan**: Generate blog topics, social media strategies, and customer review guidance
